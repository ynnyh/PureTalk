# 更新日志

记录每个版本的新增、优化和修复。版本号遵循语义化版本（major.minor.patch）。

## v0.0.3

### 新增
- **启动提示**：双击启动后自动打开设置窗口，显示"已启动"toast，首次运行显示引导
- **胶囊开关**：开机自启改为现代风格 toggle 按钮

### 修复
- **首次启动可见**：首次安装后设置窗口正确显示，不再误以为闪退
- **更新检查崩溃**：插件未启用时返回友好提示，不再关闭程序
- **toast 替代 alert**：检查更新、开机自启设置改用页面内 toast
- **构建修复**：移除 updater 签名依赖，修复 CI 构建失败
- **dev 模式跳过更新检查**：避免无 release 时报错

## v0.0.2

### 新增
- **开机自启**：设置页新增开机自启开关，基于 tauri-plugin-autostart
- **自动更新**：启动时后台检查 GitHub Releases 新版本，发现更新后提示下载安装
- **热键录制器**：点击热键区域直接录制新快捷键，支持成功/失败视觉反馈
- **官网设计**：官网全新深色紫罗兰设计，新增鸣谢 FunASR & sherpa-onnx 区块

### 优化
- **常驻语音服务**：启动时预加载 SenseVoice 模型，转写速度从首次 2-3 秒降至后续 0.5 秒内
- **智能静音裁剪**：基于能量门限自动去除录音前后静音，提升识别准确度
- **标点符号支持**：启用 SenseVoice ITN（逆文本归一化），自动添加标点符号和数字格式化
- **设置页面**：全新深色紫罗兰视觉风格，与官网统一
- **提示条窗口**：加宽至 360×64，支持流式文本预览
- **下载体验优化**：
  - 修复流式模型下载进度归零问题
  - 修复下载完成后按钮状态未刷新
  - 修正模型大小提示（437MB）
  - 移除暂不兼容的流式模型下载入口，避免用户困惑

### 技术改进
- 实现 2-pass 转写架构（常驻 offline + online WebSocket 服务）
- WebSocket 客户端支持在线流式识别（待兼容模型启用）
- 进程管理增加 Mutex 中毒恢复处理

## v0.0.1

首个公开版本。

### 新增
- **语音输入**：全局热键触发录音，再次按下停止，转写文字自动注入当前输入框
- **本地语音引擎**：sherpa-onnx + SenseVoice 离线识别，纯 CPU 亚秒级，支持中英混说，自带标点
- **云端语音引擎**：火山引擎流式 ASR，可选，更快更准
- **自定义热键**：设置内修改快捷键，支持 Ctrl/Shift/Alt/CommandOrControl 组合
- **下载管理**：语音资产流式下载，实时进度（速度/大小/百分比），断点续传，失败自动重试 3 次，支持主站 + 镜像双通道
- **下载代理**：支持配置 HTTP 代理，输入即保存，下载时自动生效
- **底部提示条**：录音/转写时屏幕底部中央显示轻量状态提示，不抢焦点，不占任务栏
- **系统托盘**：无主窗口设计，常驻托盘，右键打开设置或退出
- **首次引导**：安装后自动弹出设置窗口，引导用户配置引擎和下载资产
- **数据安全**：全部运算本地完成，无需注册登录，不收集任何数据，不开源遥测，代码完全开源可审计

### 设计
- **零界面**：应用启动后不显示任何窗口，仅系统托盘图标可见
- **不打扰**：一切交互由热键驱动，不弹窗、不提示、不询问
- **纯粹原则**：只做语音转文字，不做大模型处理，不自动发送，文字复制即用
