清语

纯粹的语音输入

Windows 上纯粹的语音输入工具。
按下快捷键，说话，松开。文字出现在你正在输入的地方，仅此而已。

下载 GitHub
Ctrl + Shift + Space
按下 → 说话 → 再按 → 完成

设计理念

零意外原则：每个功能必须是刻意的，每个操作必须是用户预期的，每个结果必须是可预测的

01

纯粹录音

一键录音，一键停止，无多余交互

02

纯粹转写

不做大模型处理，不搞"智能优化"，只做语音转文字

03

纯粹输出

文字就是文字，复制即用，不自动发送

04

不打扰

不问不答，交互由用户掌控

05

零意外

每个功能都是有意为之，每个结果都可预期

你的数据，只属于你

清语的一切运算都在你的电脑上完成。不登录、不联网、不上传，你的语音和文字永远不会离开你的设备。

无需注册

没有账号系统，没有登录流程，下载即用

完全离线

本地引擎无需联网，断网也能正常使用

零数据采集

不收集任何使用数据，没有遥测，没有埋点

本地处理

语音识别在本机完成，音频不发送到任何服务器

开源可审计

代码完全开源，任何人都可以审查每一行代码

无后台驻留

不偷偷联网，不上传后台，托盘图标是你唯一的交互入口

特性

本地优先

SenseVoice 模型本地运行，隐私安全，离线可用

极速转写

非自回归模型，纯 CPU 亚秒级识别

中英混说

自动语言检测，自带标点符号

全局热键

任意应用中一键触发，无需切换窗口

云端可选

火山引擎云端 ASR，更快更准

Windows 专属

专为 Windows 打造，解决 Windows 缺少好用语音输入工具的痛点

快速开始

1

下载安装

从 GitHub Releases 下载 Windows 安装包，双击安装

2

首次设置

安装后自动弹出设置窗口，选择引擎并下载语音资产

3

开始使用

在任意输入框中按下快捷键，说话，再次按下，文字自动输入
