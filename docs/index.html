设计理念

零意外原则

每个功能都是刻意的,每个操作都是你预期的,每个结果都是可预测的。

01

纯粹录音

一键录,一键停,没有多余交互。

02

纯粹转写

不做大模型"优化",只把语音变成文字。

03

纯粹输出

文字就是文字,复制即用,绝不自动发送。

04

不打扰

不问不答,交互完全由你掌控。

05

零意外

所见即所得,结果永远可预期。

你的数据,只属于你

所有运算都在你的电脑上完成。不登录、不联网、不上传——语音和文字永远不会离开这台设备。

快速开始

三步,开始说话

  1. 1

    下载安装

    从 GitHub Releases 下载 Windows 安装包,双击安装。

  2. 2

    首次设置

    自动弹出设置窗口,选择引擎并下载语音资产。

  3. 3

    开始使用

    任意输入框按下快捷键,说话,再按一次完成输入。
